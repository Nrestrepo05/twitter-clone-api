
# Twitter Clone

This is an API for [Twitter Clone App](https://github.com/Nrestrepo05/twitter-clone). This API was made with Node, Express, and MongoDB

## Usage

1. install the dependencies
```bash 
npm i
```
2. Create a .env file and add this:
```txt
DB_IRB = your_irb
DB_NAME = your_name
PORT = your_port
```
3. run the project
```bash 
npm run dev
```
